# _______________________________________________
## countries-large.json
Contain detailed list of countries
### format
``` bash 
country name
country code
capital city
currency { code, name, symbol }
language { code, name }
flag - link
dialing code
ISO_CODE
```

### example
``` bash
{
    "name": "Afghanistan",
    "code": "AF",
    "capital": "Kabul",
    "region": "AS",
    "currency": {
        "code": "AFN",
        "name": "Afghan afghani",
        "symbol": "Ø‹"
    },
    "language": {
        "code": "ps",
        "name": "Pashto"
}
```
